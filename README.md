# 🍕 Pizzería Mamma Mía!

Aplicación web desarrollada con **React + Vite** que simula el catálogo de una pizzería italiana. Muestra las pizzas disponibles con sus ingredientes, precios y fotos.

## Vista previa

![Pizzería Mamma Mía](https://images.unsplash.com/photo-1574071318508-1cdbab80d002?w=800&q=80)

## Tecnologías utilizadas

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- [Bootstrap 5](https://getbootstrap.com/)
- CSS personalizado

## Funcionalidades

- Navbar con botones de Login, Register y carrito de compras
- Header con imagen de portada
- Catálogo de 9 pizzas con foto, ingredientes y precio
- Footer con información de la pizzería
- Diseño responsive

## Instalación y uso

```bash
# Clonar el repositorio
git clone https://github.com/G1I2S/pizzeria-mamma-mia.git

# Entrar al directorio
cd pizzeria-mamma-mia

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.
